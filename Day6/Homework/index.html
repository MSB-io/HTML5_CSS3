🔸 📘 Key Concepts Summary Table

Property Affects Moves? Acts On Common Use
padding Inner space Inside the element Content Button sizing, text box, spacing inside cards
margin Outer space Around the element Surrounding area Pushing items apart, centering blocks
display: flex Layout Item arrangement Children of container Build horizontal/vertical responsive layouts
