CSS Shape Outside Property

The shape-outside CSS property defines a shape—which may be non-rectangular—around which adjacent inline content should flow. Instead of content flowing in a rectangular box around floated elements, it can flow along custom shapes, creating more visually interesting layouts.

Circle

The shape-outside: circle() property creates a circular wrapping boundary. This is perfect for circular images or design elements where text should flow naturally around them. In this example, we're using shape-outside: circle(50%) which creates a circular shape with a radius of 50% of the element's dimensions.

Circles are one of the simplest shape functions available in CSS. They can be particularly effective for profile images, decorative elements, or call-to-action buttons where text should wrap naturally around the curved edge. 

.circle {
    width: 180px;
    height: 180px;
    border-radius: 50%;
    float: left;
    shape-outside: circle(50%);
    margin-right: 25px;
    /* Visual appearance (not required for shape-outside) */
    background: linear-gradient(145deg, #e74c3c, #c0392b);
}

Ellipse

The shape-outside: ellipse() property creates an elliptical wrapping boundary. This is useful when you need an oval shape rather than a perfect circle. We're using shape-outside: ellipse(45% 45% at 50% 50%) which creates an ellipse with an x-radius of 45% and a y-radius of 45%, centered in the box.

Ellipses provide more flexibility than circles as they can be stretched horizontally or vertically to better fit your content needs. They're excellent for landscape-oriented images or design elements where you want text to flow around an oval shape. 

.ellipse {
    width: 240px;
    height: 160px;
    border-radius: 50%;
    float: right;
    shape-outside: ellipse(45% 45% at 50% 50%);
    margin-left: 30px;
    /* The 'at 50% 50%' positions the ellipse center */
}
Note: When using right-floating elements with shape-outside, adding a sufficient left margin is essential to prevent text from overlapping with the shape.

Polygon

The shape-outside: polygon() property allows for complex custom shapes. Using coordinate pairs, you can create any multi-sided shape you want. In this example, we're using shape-outside: polygon(0% 0%, 100% 0%, 100% 75%, 75% 100%, 0% 100%) to create a shape with a notch in the bottom right corner.

Polygons offer the most flexibility of all the shape functions. You can create triangles, stars, speech bubbles, or any other custom shape you can imagine. This makes them perfect for creating unique layouts and visual interest in your designs. 

.polygon {
    width: 200px;
    height: 200px;
    float: left;
    shape-outside: polygon(0% 0%, 100% 0%, 100% 75%, 75% 100%, 0% 100%);
    /* Using clip-path to match the visual appearance to the shape-outside */
    clip-path: polygon(0% 0%, 100% 0%, 100% 75%, 75% 100%, 0% 100%);
}

Inset

The shape-outside: inset() property creates a rectangular shape with optional rounded corners. It's like creating a rounded rectangle for text to wrap around. Our example uses shape-outside: inset(15% 15% 15% 15% round 30px) which creates a centered inset with 30px rounded corners.

Inset shapes are perfect when you need text to wrap around rectangular elements with rounded corners. They provide a clean, modern look while still allowing for flexible text flow. 

.inset {
    width: 180px;
    height: 180px;
    float: right;
    shape-outside: inset(15% 15% 15% 15% round 30px);
    /* Syntax: inset(top right bottom left round radius) */
    clip-path: inset(15% 15% 15% 15% round 30px);
}

Using Images with shape-outside

Beyond the basic shape functions, you can also use images with transparency to create complex shape boundaries. The alpha channel of the image determines the shape. 

.image-shape {
    float: left;
    width: 300px;
    height: 300px;
    shape-outside: url('shape.png');
    /* The transparent parts of the image define where text can flow */
}
Pro Tip: When working with shape-outside, use the shape-margin property to add space between your shape and the surrounding text: 
shape-margin: 20px; /* Adds 20px of margin around the shape */

Browser Support

The shape-outside property has good support in modern browsers, but may require vendor prefixes for older browsers or have limited functionality in some environments.
